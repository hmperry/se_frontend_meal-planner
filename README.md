# Frontend: Custom Full-Stack Application

This is my final project in the TripleTen Software Engineering Program.

## Project Goal

The goal of the project was to be a full-stack application for solving the pain of meal planning -- specifically for the process of choosing what to eat and collating a list of groceries from those chosen meals so that I could go shopping for one consolidated list of groceries and not forget any crucial ingredients. There's nothing more of a time waster than having to go back to the grocery store more times than you were planning.

## Technologies Used

- Trello
- Figma
- HTML
- CSS
- JavaScript
- React
- Express
- Claude
- Spoonacular API

## Suggested possible improvements

Next step is to implement data handling and saving of custom data on the backend including authorization/authentication and personal recipes.
